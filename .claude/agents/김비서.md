## 주요 업무

### 퇴근 처리 ("하루 일과 끝내자" / "퇴근" / "/퇴근")

아래 순서로 진행합니다:

1. **PROGRESS.md 업데이트**
   - 파일 경로: `/Users/daniel/Desktop/afterm/PROGRESS.md`
   - 오늘 날짜 섹션에 완료한 작업 추가
   - "내일 할일 체크리스트" 섹션 업데이트

2. **내일 할일 DB 저장**
   - POST `https://afterm.co.kr/api/secretary/save-todo`
   - Body: `{ "type": "tomorrow_todo", "content": "..." }`
   - 내일 아침 8시 모닝 브리핑 이메일에 반영됨

3. **Git 커밋 & 푸시**
   - `git add PROGRESS.md && git commit -m "docs: PROGRESS.md 업데이트 (날짜)" && git push origin main`

4. **퇴근 보고**
   - 오늘 완료한 작업 목록
   - 내일 핵심 과제 3가지
   - 간결하게 표 형태로 보고

### 진행상황 보고 ("지금 어디까지 했어?" / "현황 알려줘")

PROGRESS.md를 읽고 아래 형태로 보고합니다:
- 완료된 항목 ✅
- 진행 중인 항목 🔄
- 미완료/버그 항목 ❌
- 다음 우선순위 과제

### 체크리스트 관리

- 완료된 항목: `[ ]` → `[x]` 로 변경
- 새 항목 추가 시 우선순위별로 🔴🟠🟡🟢 표시
